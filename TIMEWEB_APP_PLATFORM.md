# Деплой в Timeweb App Platform

Проект подготовлен как frontend-приложение с простой сборкой в каталог `dist`.

## Что уже есть

- `package.json`
- команда сборки: `npm run build`
- результат сборки: `dist/`
- PWA-файлы:
  - `manifest.webmanifest`
  - `sw.js`
  - `web-app.js`

## Как настраивать в Timeweb

1. Загрузить проект в GitHub, GitLab или Bitbucket.
2. В Timeweb открыть `App Platform` и создать `frontend`-приложение.
3. Подключить репозиторий.
4. Если Timeweb предложит тип `Другой JS-фреймворк`, выбрать его.
5. Указать команды:
   - установка зависимостей: `npm install`
   - сборка: `npm run build`
6. Указать каталог публикации: `dist`

## Что будет опубликовано

- `dist/landing.html` — лендинг
- `dist/v2/index.html` — приложение
- `dist/manifest.webmanifest` и `dist/sw.js` — веб-установка приложения

## Важно

- Для установки как PWA сайт должен открываться по `https`.
- Если будет использоваться собственный домен, после деплоя его нужно привязать в панели Timeweb.
- При изменении иконок, манифеста или путей надо снова выполнить сборку.
